# 🐍 100 Days of Python

A follow-along repo for the Udemy course [**100 Days of Code: The Complete Python Bootcamp**](https://www.udemy.com/course/100-days-of-code/) by Dr. Angela Yu.

---

## 📌 About

This repository contains my final project submissions for each day of the course. Each project is self-contained and represents the capstone exercise for that day's material — covering everything from beginner fundamentals to intermediate and advanced Python concepts.

---

## 📂 Structure

```
100-days-of-python/
├── day-001/
├── day-002/
├── ...
└── day-100/
```

Each folder is named by day and contains the final project for that lesson.

---

## 🗺️ Course Roadmap

| Range | Level | Topics Covered |
|-------|-------|----------------|
| Days 1–14 | Beginner | Variables, data types, control flow, loops, functions |
| Days 15–31 | Intermediate | OOP, files, error handling, APIs, web scraping |
| Days 32–57 | Intermediate+ | GUI, data science, web development (Flask) |
| Days 58–80 | Advanced | REST APIs, authentication, databases, deployment |
| Days 81–100 | Professional | Portfolio projects, automation, data structures |

---

## 🛠️ Requirements

- Python 3.x
- Dependencies vary by project — check individual day folders for any `requirements.txt`

## 🚀 Getting Started

```bash
git clone https://github.com/CyberSteveon/100-days-of-python.git

```
